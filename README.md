# Energy-Peak-Risk

## Project Overview

This project predicts household electricity demand using Machine Learning. The objective is to identify future electricity demand and support peak-load risk management.

## Problem Statement

Electricity demand changes throughout the day. Predicting future demand helps reduce peak load, improve energy planning, and support efficient power management.

## Machine Learning Task

- **Task:** Regression
- **Target Variable:** Global_active_power

## Dataset

- **Name:** Individual Household Electric Power Consumption Dataset
- **Source:** UCI Machine Learning Repository

## Models Used

- Dummy Regressor
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

## Best Model

Gradient Boosting Regressor achieved the best performance based on R² score.
