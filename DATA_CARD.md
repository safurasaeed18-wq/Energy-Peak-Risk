# DATA CARD

## Dataset Name

Individual Household Electric Power Consumption Dataset

## Source

UCI Machine Learning Repository

## License

Open dataset for research and educational purposes.

## Problem Type

Regression

## Target Variable

Global_active_power

## Number of Records

2,075,235 rows

## Features

- Date
- Time
- Global_reactive_power
- Voltage
- Global_intensity
- Sub_metering_1
- Sub_metering_2
- Sub_metering_3
- Derived date and time features

## Missing Values

Missing values represented by "?" were converted to NaN and handled during preprocessing.

## Data Usage

This dataset is used to predict household electricity consumption and identify peak electricity demand.

## Limitations

- Data represents a single household.
- Weather information is not included.
- Results may not generalize to all buildings.
